# typescrippt

```
yarn add -D typescript @types/react @types/react-dom @types/node
```

# axios

```
yarn add axios
```

# 拡張子変更

JSX 使っているなら拡張子を"tsx"に変更する
使っていない場合は"ts"に変更する

# 開発サーバ起動

`yarn dev`

# 型の定義

エラーになった箇所に対して、型を定義する

# function と定義されている箇所で戻り値の型定義を指定する場合はアロー関数に変更する

# ビルドする

`yarn build`
